# MindForge Mobile App

Mobile companion app for the **MindForge** project.

This repository contains the Android client built with React Native + Expo (Bare Workflow). It connects to the main MindForge backend and supports quiz solving, practice, favorites, creator flows, and offline usage for cached content.

## Main Project

- Main repository: [MindForge](https://github.com/RiscyX/MindForge)
- Mobile repository: [MindForgeMobileApp](https://github.com/RiscyX/MindForgeMobileApp)

## Highlights

- Quiz types: single choice, text, matching
- Practice mode and regular attempts
- Favorites with offline cache
- Offline result queue with auto-sync on reconnect
- AI-assisted quiz creation (creator role)
- EN/HU language support
